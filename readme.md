# PostbirdAutoCheckIn

> 直播平台弹幕自动签到工具

> 在线使用：https://postbird.gitee.io/postbirdautocheckin/

## 作用范畴：

适用于国内直播平台通过弹幕发布控制命令的工具，如“弹幕打卡”等

## 作用平台：

直播平台PC网页端

## 更新中：

目前支持`pandan.tv`弹幕自动打卡等，后续平台追加更新中

## 文档：

https://postbird.gitee.io/postbirdautocheckin/
